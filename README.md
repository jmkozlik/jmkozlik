### [Agent CRM](https://github.com/MediaInfluences/agent-crm)

An autonomous AI-agent-operated sales system. The agents run the complete pipeline — deciding who to contact, what to say, when to follow up, and how to handle objections.

**The agents don't assist salespeople. They _are_ the salespeople.** Humans confirm outbound emails, execute live calls, and provide the final word — but the thinking belongs to the agents.

`Python 3.12` `FastAPI` `React 19` `Vite` `PostgreSQL` `Deepgram`

## `> philosophy`

We believe AI agents should be more than chatbots with API keys.
They should have **memory**, **judgment**, and **autonomy** — with humans in the loop for the decisions that matter.

The boulder is heavy. The hill is steep. But somewhere between the pushing and the rolling back, we find the work worth doing.
